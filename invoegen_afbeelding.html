Invoegen afbeelding

Het toevoegen van plaatjes op een webpagina

Stap 1

Het is een good practice om afbeeldingen bestemd voor een webpagina op te slaan in een aparte map. Indien dit nog niet gebeurt is: Maak een folder 'static' in de directorie waar de html pagina is gemaakt.

Stap 2

Sla de afbeelding op in de map static.

Stap 3

Verwijs in de code van de html pagina naar de te gebruiken afbeelding met de optie img. Hieronder staat de gebruikte code weergeven. Deze code dient in de body van een html pagina op de gewenste plek gezet te worden.

Code 

                            <img src="static/bootstrap5.jpg" alt="bootstrap logo" class="w-50">
Resultaat


Stap 4

Bootstrap heeft de mogelijkheid om de afbeeldingen te bewerken. Zo is het mogelijk om rand van de afbeelding te bewerken met de opties: rounded, rounded-circle of thumbnail. Voeg deze simpelweg toe aan de class attrbuut van de afbeelding.

rounded
rounded-circle
thumbnail




Uiteraard zijn er meer opties beschikbaar met de menubalk. Maar die vallen buiten de scope van deze cursus.

Validatie

Stap 1

Het is mogelijk om een validatieknop te maken met het type checkbox van de optie form-check-input. Om knoppen als de validatieknop te gebruiken zijn formulieren nodig die verstuurd kunnen worden. Daarom wordt de klasse form-check gebruikt. Hieronder staat de gebruikte code voor een standaard validatieknop waarbij 1 knop al geselecteerd is en de ander niet. Voeg deze code toe in de body van een html document om die toe te voegen aan de html-pagina.

Code 

                            <div class="form-check mb-3">
                                <br>
                                <label class="form-check-label">
                                    <input class="form-check-input" type="checkbox" name="knop1" checked> Validatieknop 1
                                </label>
                                <br>
                                <label class="form-check-label">
                                    <input class="form-check-input" type="checkbox" name="knop2"> Validatieknop 2
                                </label>
                            </div>
Resultaat



Het maken van een formulier die vervolgens gebruikt kan worden binnen de pagina, bijvoorbeeld voor het opslaan van gebruikersinstellingen in een database, vallen buiten de scope van deze cursus.

Drop-down

Stap 1

Het is mogelijk om een drop-down keuze menu te maken met de optie form-select. Om de drop-downkeuze menu selecties op te slaan zijn formulieren nodig die verstuurd kunnen worden. Hieronder staat de gebruikte code voor een standaard drop-downkeuze menu van instrumenten. Voeg deze code toe in de body van een html document om die toe te voegen aan de html-pagina.

Code 

                            <select class="form-select" id="instrument" name="instrumentenlijst">
                                <option selected>Kies hier de instrument die je kan bespelen</option>
                                <option>Piano</option>
                                <option>Gitaar</option>
                                <option>Contrabas</option>
                                <option>Viool</option>
                                <option>Ik kan geen instrument bespelen</option>
                            </select>
Resultaat


Te zien is dat elke mogelijke selectie binnen option html-tags staan. De lijst kan willekeurig lang worden. Het maken van een formulier die vervolgens gebruikt kan worden binnen de pagina, bijvoorbeeld voor het opslaan van gebruikersinstellingen in een database, vallen buiten de scope van deze cursus.

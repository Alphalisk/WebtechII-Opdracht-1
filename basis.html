Basis

Stap 1

Om een website te maken met bootstrap features is het allereerst handig om over een basis html pagina te beschikken. Binnen de cursus wordt ervan uit gegaan dat de cursist basiskennis in html en css beheerst.
Open een teksteditor als notepad, op de computer en sla de volgende code op als een .html bestand. Dit is de code van een lege html pagina met als titel 'demo' en een paragraaf binnen een container met de tekst Hello world. Bootstrap werkt erg goed met containers om de pagina in stukken te delen en tekst uit te lijnen.

Code 

                            <!DOCTYPE html>
                            <html lang="en">
                                <head>
                                    <title>Demo</title>
                                    <meta charset="utf-8">
                                </head>
                                <body>
                                    <div class="container">
                                        <p>
                                            Hello world
                                        </p>
                                    </div>
                                </body>
                            </html>
Resultaat


Stap 2

Op dit moment wordt er nog geen gebruik gemaakt van bootstrap. Door de volgende lijnen met code toe te voegen aan de header van het document wordt er een link gemaakt met bootstrap versie 5.

Code 

                            <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
                            <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
Resultaat


Het resultaat is subtiel, maar te zien is dat het lettertype en de uitlijning van de tekst Hello world met bootstrap anders is geworden dan bij het eerste resultaat. Door gebruik te maken van bootstrap zal de opmaak van de pagina gestandaardiseerd worden.
